## How to Start
1. Clone the project
2. Import the Project in PyCharm
3. Run `robot -d Results Tests/TestName.robot` to execute a single Test. Run `robot -d Results Tests` to execute all Tests.
4. Go to `Results` folder to get the HTML Reports.
